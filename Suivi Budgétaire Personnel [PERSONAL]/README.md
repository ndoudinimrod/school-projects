# 💰 Suivi Budgétaire Personnel — Personal Budget Tracker

> Tableaux Excel interactifs (mensuels & annuels) pour la gestion des finances personnelles

---

## 🇬🇧 Overview

Interactive Excel spreadsheets (monthly & annual) designed to help users manage their personal finances — both **manually and automatically** from bank statements.

## 🇫🇷 Description

Des **tableaux Excel interactifs** (mensuels et annuels) conçus pour aider à gérer efficacement les finances personnelles, manuellement ou **automatiquement à partir de relevés bancaires**.

---

## ⚙️ Fonctionnalités

- 📥 Import automatique depuis relevés bancaires
- 📊 Suivi des revenus et dépenses
- 📈 Analyse des tendances budgétaires dans le temps
- 🧮 Formules avancées et automatisation
- 📉 Visualisation claire des données financières
