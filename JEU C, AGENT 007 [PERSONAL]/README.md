\# 🕵️ Agent 007 — Jeu de déduction en langage C

\## 🇬🇧 Overview



A \*\*deduction game written in C\*\* where you must guess a secret combination randomly generated by the computer.  

Features a multiplayer mode, a secret mode... and a hidden menu only Agent 007 knows about. 🤫  

> ⚠️ Warning: if you lose, a bomb might shut EVERYTHING down. 💥



\## 🇫🇷 Description



Un \*\*jeu de déduction en langage C\*\* où vous devez deviner la combinaison secrète générée aléatoirement par l'ordinateur.  

Mode multijoueur disponible, mode secret, et un menu caché réservé à l'agent 007. 🤫
