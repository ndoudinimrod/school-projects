# 🐾 Pokédex App — Application Android Kotlin



> Application Android permettant de visualiser tous les Pokémon officiels, de la 1ère à la dernière génération



---



## 🇬🇧 Overview



An **Android application developed in Kotlin** (1 week) that displays all official Pokémon from Generation 1 to the latest, in both **normal and shiny** forms. This entire application is based on the API: https://pokeapi.co/api/v2 .



## 🇫🇷 Description



Application **Android développée en Kotlin** en 1 semaine permettant de visualiser tous les Pokémon officiels de la licence Pokémon — de la 1ère génération à la dernière — en mode **naturel et shiny**. Tout cette application est basé sur l'API : https://pokeapi.co/api/v2 .



---



## ⚙️ Fonctionnalités actuelles



- 📋 Liste complète de tous les Pokémon officiels

- ✨ Visualisation en mode **naturel** et **shiny**

- 🔍 Navigation par génération
