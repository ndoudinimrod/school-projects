\# 🃏 BlackJack — Jeu en Java



> Jeu de BlackJack développé en Java avec système de sauvegarde des scores



\---



\## 🇬🇧 Overview



A \*\*BlackJack game developed in Java\*\* as a school project, featuring a highscore system that saves each player's bank and user scores in a dedicated file named after the user.



\## 🇫🇷 Description



Jeu de \*\*BlackJack développé en Java\*\* dans le cadre d'un projet scolaire à l'ESIEA, avec un système de sauvegarde des meilleurs scores par joueur.



\---



\## ⚙️ Fonctionnalités



\- 🃏 Jeu de BlackJack complet (règles classiques)

\- 🏆 Sauvegarde des \*\*Highscores\*\* par joueur

\- 📁 Création automatique d'un fichier par utilisateur contenant :

&#x20; - Score de la \*\*banque\*\*

&#x20; - Score du \*\*joueur\*\*
