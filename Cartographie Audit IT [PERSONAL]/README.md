# 🗺️ Cartographie Audit IT — Risk Mapping Tool

> Tableur Excel interactif pour la cartographie des risques des Systèmes d'Information (SI)

---

## 🇬🇧 Overview

Development of an **interactive Excel risk mapping tool** for Information Systems (IS).  
The application allows users to identify, assess and visualize risks related to IT assets through a user-friendly interface, with report generation and graphical data visualization.

## 🇫🇷 Description

Développement d'un **tableur Excel interactif** pour la cartographie des risques des systèmes d'information (SI).  
L'application permet aux utilisateurs d'identifier, d'évaluer et de visualiser les risques liés aux actifs informatiques, avec gestion des utilisateurs, génération de rapports et visualisation graphique.

---

## ⚙️ Fonctionnalités

- 🗂️ Identification et évaluation des risques IT
- 📊 Visualisation graphique des données de risque (matrices, graphiques)
- 👥 Gestion multi-utilisateurs
- 📄 Génération de rapports
- 🔄 Interface conviviale et intuitive
